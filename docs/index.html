What's included

Everything you need in an SSH client

⌨️

Full terminal emulation

VT100 / xterm-compatible terminal with multiple simultaneous sessions.

🔑

Credentials vault

Store SSH keys and passwords with Android Keystore encryption. Keys never leave your device in plaintext.

⚡

Quick Commands

Save, organise, and fire off commands instantly from a toolbar above the keyboard.

🔐

SSH key management

Generate RSA 2048/4096 and Ed25519 keys, or import from file. Link keys to connections for auto-auth.

🔁

Port forwarding

Local, remote, and dynamic (SOCKS5) port forwarding support built in.

💾

Encrypted backup

Export and import all your data with an optional encrypted .sapienterm backup file.

🛡️

Host key verification

Fingerprint display and strict host key checking protect you from MITM attacks.

🎨

Material Design 3 UI

Card-based connection management with status indicators, dark and light themes.

📡

Keep-alive & reconnect

Configurable keep-alive pings and network-change reconnect keep sessions stable on mobile.
