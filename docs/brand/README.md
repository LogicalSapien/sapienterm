# SapienTerm Brand Assets

Three logo concepts. Primary (used in launcher) is **Wise Arrow**.

| Concept | Vector | SVG source | Used in |
| --- | --- | --- | --- |
| Wise Arrow | `app/src/main/res/drawable/ic_sapien_wise_arrow.xml` | `sapien-wise-arrow.svg` | launcher + in-app icon |
| Connected Mind | `app/src/main/res/drawable/ic_sapien_connected_mind.xml` | `sapien-connected-mind.svg` | alternative |
| Sapien Sigil | `app/src/main/res/drawable/ic_sapien_sigil.xml` | `sapien-sigil.svg` | alternative |

Accent color is `#A6FF7A` (Neo-Terminal lime). For Quiet Luxury use `#7FB89A`; for Soft Pro use `#7C3AED`.

The launcher uses `ic_launcher_foreground.xml` (Wise Arrow) on an `ic_launcher_background.xml` solid near-black background, exposed via `mipmap-anydpi-v26/icon.xml` and `icon_round.xml` (referenced by `AndroidManifest.xml` as `@mipmap/icon`).
